# Bank Loan Analysis Dashboard

## Overview

This project focuses on analyzing loan portfolio data using SQL, Python, and Power BI to evaluate lending performance, customer risk indicators, and financial KPIs. The objective is to transform raw loan data into actionable business insights through data analysis and interactive dashboards.

## Tools & Technologies

* SQL
* Python (Pandas, NumPy)
* Power BI
* Excel

## Key KPIs

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Debt-to-Income (DTI) Ratio
* Good Loans vs Bad Loans

## Analysis Performed

* Loan Status Analysis
* Regional Analysis by State
* Loan Purpose Breakdown
* Home Ownership Analysis
* Credit Risk Assessment
* Portfolio Performance Monitoring

## Dashboard Features

* Interactive KPI Tracking
* Loan Performance Monitoring
* Regional Lending Insights
* Good vs Bad Loan Comparison
* Customer Risk Analysis

## Business Insights

The project helps identify lending trends, monitor portfolio performance, evaluate customer risk, and support data-driven financial decision making.
